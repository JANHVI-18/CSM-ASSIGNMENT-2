Welcome to My Cloud Project

This project is part of the CSM ASSIGNMENT, where I'll be exploring and implementing various cloud concepts and operations every week.

Project Overview

The project involves deploying a scalable and secure web application on AWS using a three-tier architecture. The setup includes Virtual Private Cloud (VPC) for networking, an application load balancer, EC2 instances for the application layer, and an RDS database for storage.

Goals for Week 1

This week’s focus is on setting up the project foundation, including environment setup, VPC creation, and initial EC2 instance configuration. Additionally, I'll document each step, making it easier to track progress and ensure reproducibility.
